# Пуррчик (Purrchik)

ВК бот для записи на маникюр в салоне "ForPoor"

## Установка
1.[Установить node.js версии 14.15.2 или выше](https://nodejs.org/en/ "node.js")

2. Ввести токен вашего соообщества ВК в token.js
3. Установить зависимости
```
npm install
```

## Запуск
```
npm start
```
или
```
node ./index.js
```
